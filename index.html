Sales Performance Dashboard

Context
BikeGear Solutions required clear visibility into revenue, cost, profit, and overall sales performance across regions, customers, and products.

What I worked on
I began by cleaning and structuring the raw sales data to ensure accuracy and consistency. Customers were grouped into defined customer segments, after which key measures such as revenue, cost, and profit were calculated. With a reliable dataset in place, I built pivot tables to analyse performance by time period, product, region, and customer group, and used these insights to design an interactive Excel dashboard.

Focus areas
Data preparation, customer grouping, KPI calculation, performance analysis, and dashboard design.

Tools
Excel (data cleaning, PivotTables, charts), basic data modelling.

Hospital Operations Dashboard (Power BI)

Context and objective
This project involved cleaning and modelling raw hospital operational data to produce an interactive Power BI dashboard that supports stakeholder decision-making on patient activity, provider workload, services delivered, and performance trends over time.

Data used
Source: Raw hospital dataset in Microsoft Excel.
Content: Transaction-level hospital records (patient, doctor, procedure, location, date fields, plus activity metrics where applicable).

What was done (high level)

  • Data ingestion: Imported and loaded Excel data into Power BI.
  • Data cleaning and preparation (Power Query): Removed null/blank values where appropriate, removed duplicates, merged tables where needed, and removed staging tables not required for the final model.
  • Data modelling: Built a star schema with a central fact table (hospital transactions) and supporting dimension tables (Patient, Doctor, Date, Location, Procedure). Created one-to-many relationships using key fields (PatientID, DoctorID, LocationID, ProcedureID, and Date key).
  • DAX calculations: Created measures for key metrics and comparisons across time, doctors, patients, procedures, and locations.
  • Reporting and visualisation: Built the dashboard layout to communicate trends and operational insights clearly for stakeholders.

Tools
Power BI Desktop, Power Query, DAX, Excel.
